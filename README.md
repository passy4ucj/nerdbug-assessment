

# Nerdbug

User microservice

## Prerequisites

Make sure you have the following installed on your machine:

- [Node.js](https://nodejs.org/)
- [npm](https://www.npmjs.com/)
- [MySQL](https://www.mysql.com/)

## Getting Started

1. **Clone the Repository:**
   git clone https://github.com/passy4ucj/nerdbug-assessment
   cd nerdbug-assessment
### Install Dependencies:


- `npm install``

### Set Up Database:

- Create a MySQL database for your microservice.
Update the `.env.example` to `.env`

Build and Start:

If your project is in TypeScript, build it:

- `npm run build`

Start the microservice:

- `npm run dev`
Verify:

Open your web browser and go to http://localhost:5000/api/v1.
Testing
Run tests with the following command:

npm test
Additional Information

### DOCKER

- RUN `docker-compose start`
